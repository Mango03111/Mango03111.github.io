Typora 是一款跨平台的 Markdown 文本编辑器，其最大的特点是“**所见即所得（WYSIWYG）**”的编辑体验。这意味着当你在 Typora 中输入 Markdown 语法时，它会在同一界面实时渲染、显示最终效果，而不是像其他很多编辑器那样需要分别显示源码窗口和预览窗口。

**主要优点包括：**

- **实时预览**：无需切换视图，即时看到排版后的效果。
- **界面简洁**：专注内容编辑，不受复杂 UI 干扰。
- **支持丰富导出格式**：如 PDF、HTML、Word 等，方便分享与发布。
- **主题与样式多样化**：可根据喜好自定义外观。
- **跨平台支持**：可用于 Windows、macOS、Linux 等系统。

这种编辑模式使 Typora **既适合写博客、笔记，也适合编写技术文档与代码说明**。

------

## 准备工作

在正式安装之前，你需要准备：

1. **Typora 官方安装包**（建议选择 1.9.5 版本，因为后续版本可能需要付费或不兼容免费激活工具）
2. **辅助工具包**（包含两个可执行文件：`node_inject.exe` 和 `license-gen.exe`），用于生成激活序列号。

⚠️ 注意：这些辅助工具来源于开源项目，**仅限学习与技术交流使用**。在使用时请保证来源合法且不要用于商业传播。

------

## 详细安装步骤

下面是逐步说明每一个操作及其作用：

------

### **步骤 1 — 下载必要文件**

你需要分别下载：

**Typora 官方安装包**：可以从官网或可信任的镜像站获取安装程序，在历史版本中选择1.9.3版本（解压后会更新为1.9.5版本）。

Typora 中文官网地址：[Typora中文网](https://typoraio.cn/)

**激活辅助工具包**：下载解压后会得到两个文件：

- `node_inject.exe`
- `license-gen.exe`

网盘分享的链接：[下载 Typroa_Activation_Script](https://pan.baidu.com/s/1dkg9FMa77zXLRcenipHAhw?pwd=s3w3)

👉 将下载的文件解压并准备好相关工具。

------

### **步骤 2 — 安装 Typora**

双击运行 Typora 安装程序，按照安装向导提示一步一步进行安装：

- 选择安装路径（建议默认安装路径或者自定义路径）
- 点击“下一步”
- 等待安装完成

👉 这一步只是将 Typora 安装到电脑上，并**不运行它（注意在点finish之前取消勾选运行框）**。

------

### **步骤 3 — 准备激活工具**

将下载并解压出的两个激活工具文件：

```
node_inject.exe
license-gen.exe
```

复制到你刚才安装 Typora 的目录下。例如默认路径可能是：

```
C:\Program Files\Typora
```

如果你安装到其他盘（如 D 盘），路径可能类似：

```
D:\Program Files\Typora
```

👉 将程序和工具放在同一文件夹有利于后续命令执行。

------

### **步骤 4 — 打开管理员命令行**

为了运行激活工具，你需要启动 **命令提示符（CMD）** 并以管理员身份运行：

1. 按下 **Windows** 键
2. 搜索：`CMD`
3. 右击“命令提示符”，选择“**以管理员身份运行**”

👉 以管理员身份运行是为了保证激活工具有足够权限写入相关信息。

------

### **步骤 5 — 生成激活序列号**

在管理员 CMD 中，输入如下命令：

1. 跳转到 Typora 安装目录：

   ```
   cd C:\Program Files\Typora
   ```

2. 运行第一个工具：

   ```
   node_inject.exe
   ```

   等待命令行提示 “done!” 说明注入成功。

3. 再运行生成序列号的工具：

   ```
   license-gen.exe
   ```

   等待程序显示生成的序列号。

   ![截图20260202235740](截图20260202235740.png)

👉 这一步是关键：它会生成一个可以用于激活 Typora 的有效序列号。

------

### **步骤 6 — 激活 Typora**

现在打开 Typora：

1. 输入一个任意邮箱地址（如 [test@example.com](mailto:test@example.com)）
2. 将刚才生成的序列号粘贴到激活界面
3. 点击确认完成激活

![截图20260202235630](截图20260202235630.png)

![截图20260202235722](截图20260202235722.png)

如果一切配置没有问题，你应该会看到 **Typora 已被成功激活** 的提示。

------

### **步骤 7 — 异常处理提示**

激活过程若出现错误提示：

- 可能是网络原因（工具试图联网验证）

一般情况下，只需点击确认继续即可解决多数错误提示。

![截图20260202235707](截图20260202235707.png)

------

### **步骤 8 — 安装完成注意事项**

- **完成后请不要自动更新 Typora**，更新可能导致激活失效。
- 若升级需要重新激活，可重复以上步骤。

🎉 至此，你就成功安装并激活了 Typora，可以开始流畅写 Markdown 文档啦！
